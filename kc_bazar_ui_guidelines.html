KC Bazar UI Guidelines

1. Context and Goals

Mission: Create implementation-ready, token-driven UI guidance for KC Bazar that is optimized for consistency, accessibility, and fast delivery across the e-commerce storefront.

Design Intent: To provide a structured, content-first, and highly accessible e-commerce experience for online shoppers that feels confident, reliable, and premium.

2. Design Tokens and Foundations

All components must utilize the semantic tokens defined below. Raw hex values and one-off pixel values are strictly prohibited.

Typography

  • Primary Font: font.family.primary = Noto Sans Bengali
  • Font Stack: font.family.stack = Noto Sans Bengali, Arial, Helvetica, sans-serif
  • Base Style: font.size.base = 16px | font.weight.base = 400 | font.lineHeight.base = 25.6px
  • Typography Scale: xs=12px, sm=12.8px, md=13px, lg=14px, xl=14.88px, 2xl=15px, 3xl=16px, 4xl=16.32px

Color Palette

  • Text: primary = #333333, secondary = #242424, tertiary = #838383, inverse = #767676
  • Surface: base = #000000, muted = #e2528c Primary Brand, raised = #f3f3f3, strong = #ffffff
  • Border: muted = rgba(0, 0, 0, 0.106)

Spacing Scale & Radii

  • Spacing: space.1 = 5px, space.2 = 8px, space.3 = 10px, space.4 = 12px, space.5 = 14px, space.6 = 15px, space.7 = 18px, space.8 = 20px
  • Radii: xs = 5px, sm = 35px, md = 50px
  • Shadow: shadow.1 = rgba(0, 0, 0, 0.17) 0px 0px 5px 0px

4. Accessibility Requirements

  • Target: WCAG 2.2 AA compliance is mandatory.
  • Keyboard-First: All interactive elements must be fully functional using only a keyboard (Tab, Space, Enter).
  • Focus-Visible: Custom focus rings must be explicitly defined using :focus-visible.
  • Contrast: Text must meet a minimum contrast ratio of 4.5:1 against its background.
